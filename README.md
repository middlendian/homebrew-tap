# Middlendian Tap

## How do I install these formulae?

`brew install middlendian/tap/<formula>`

Or `brew tap middlendian/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "middlendian/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
